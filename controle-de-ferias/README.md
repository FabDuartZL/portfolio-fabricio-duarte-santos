# 🏖️ Sistema de Controle de Férias

## 📝 Descrição do Projeto

O **Sistema de Controle de Férias** é uma plataforma web desenvolvida para automatizar e facilitar a gestão de férias de colaboradores em empresas e organizações. O sistema centraliza solicitações, aprovações e acompanhamento de períodos de férias, proporcionando maior organização, transparência e eficiência para os setores de Recursos Humanos e gestores.

Desenvolvido utilizando a plataforma **Bubble.io**, o projeto oferece uma solução moderna baseada em nuvem, permitindo gerenciamento completo dos períodos aquisitivos, saldo de férias, calendário de afastamentos e fluxo de aprovações em tempo real.

A aplicação busca reduzir erros operacionais, otimizar processos administrativos e garantir melhor controle das escalas de férias dos colaboradores.
